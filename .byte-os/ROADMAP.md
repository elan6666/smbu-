# Roadmap

## v0 Deliverable

- Initialize GitHub-connected repository.
- Build source crawler for official SMBU admissions pages.
- Store cleaned documents and metadata.
- Build structured admissions-score table schema.
- Implement RAG retrieval and grounded answer generation.
- Implement score-query endpoint and table rendering.
- Implement Web chat demo.
- Add lightweight intent classifier training and evaluation.
- Add evaluation set and report-ready result export.
- Deploy on server and document run commands.
- Write LaTeX Chinese report and compile PDF.

## v1 Improvements

- Add more years of admissions data and province coverage.
- Add reranker ablation: dense retrieval only vs dense+rerank.
- Add applicant risk bands using historical score/rank distance, with conservative disclaimers.
- Add better source browsing UI.
- Add demo recording and screenshots for submission.

## Later Opportunities

- Multi-school comparison.
- Parent/applicant persona modes.
- Automatic yearly data refresh.
- Better bilingual Chinese-English/Russian support.
- More formal answer faithfulness evaluation.
