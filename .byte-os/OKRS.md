# OKRs

## Objective

Deliver a complete, reproducible NLP course project: a Shenzhen MSU-BIT University admissions dialogue system with Web demo, server deployment, lightweight model training, evaluation, and a Chinese LaTeX report.

## Key Results

- KR1: Index official admissions and school information sources with source metadata and update timestamps.
- KR2: Support structured score lookup by year, province, category, admission mode, and major where data is available.
- KR3: Run at least 50 evaluation questions and export report-ready results.
- KR4: Train and evaluate a lightweight intent/query-routing model with accuracy and macro-F1 reported.
- KR5: Deploy a runnable Web demo on the server and capture logs/screenshots.
- KR6: Compile the Chinese LaTeX report to PDF with real references and no fabricated claims.

## Quality Bar

- Prefer a smaller but complete system over a large but fragile prototype.
- Every major result in the report must be backed by code output, logs, screenshots, or saved data.
- Every cited source must be real and checked before final submission.
